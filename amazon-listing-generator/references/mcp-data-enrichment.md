# MCP Data Enrichment

Use this only when Amazon data tools such as Sorftime, SellerSprite, or similar MCP sources are available.

## Use Data For

- Keyword pool expansion
- Competitor title and bullet pattern analysis
- Review pain point discovery
- Category positioning
- Compatibility and risk checks

## Safety Boundary

- MCP data is supporting evidence for internal decisions.
- Do not write sales, BSR, PPC, review ratios, or competitor metrics as shopper-facing claims.
- If MCP data conflicts with user-provided official product facts, ask the user to confirm before using it.
- Keep keyword and review conclusions factual and conservative.
